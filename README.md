# Vandu's Bar - O Verdadeiro Kibe de Formosa

Site oficial do Vandu's Bar, um bar/restaurante brasileiro em Formosa, Goiás. Especializada em kibes e petiscos há mais de 24 anos.

## Tecnologias

- HTML5
- CSS3 (vanilla)
- JavaScript (ES6+)
- Google Fonts (Lato)
- Font Awesome CDN

## Como Executar

Não há processo de build. Basta abrir o arquivo `index.html` em um navegador.

### Desenvolvimento Local

```bash
# Usando Python
python -m http.server 8000

# Usando Node.js
npx serve
```

## Estrutura do Projeto

```
vandus-bar/
├── index.html      # Estrutura HTML principal
├── styles.css     # Estilos CSS
├── logic.js       # Lógica JavaScript
├── images/        # Imagens do projeto
├── AGENTS.md      # Diretrizes para agentes
└── README.md      # Este arquivo
```

## Seções do Site

1. **Navegação** - Menu fixo com links para seções
2. **Hero** - Seção principal com imagem de fundo
3. **Sobre** - História do Vandu's Bar
4. **Especiais da Semana** - Destaques semanais
5. **Cardápio** - Cardápio completo com abas
6. **Galeria** - Fotos do espaço
7. **Localização** - Mapa e informações de contato
8. **Contato** - Formulário de contato
9. **Footer** - Links e informações legais
